⚠ GitHub Pages Setup — Do This First

  1. Create a GitHub repo (e.g. ar-menu)
  2. Upload both index.html and ar-viewer.html to the repo root
  3. Create a folder called models/ and upload your .glb / .gltf files there
  4. Go to Settings → Pages → Source: main branch → Save
  5. Your base URL will be: https://YOUR-USERNAME.github.io/ar-menu/
  6. Enter your GitHub Pages URL below, then add dishes
